Step 1: Choose Your Subdomain

Step 2: GitHub Repository Setup

Follow these steps to set up your GitHub repository:

  • Create a new repository on GitHub
  • Name it: yourusername.github.io
  • Upload your website files:
    • index.html (required)
    • style.css (if you have styling)
    • script.js (if you have JavaScript)
    • Other assets (images, videos, etc.)
  • Enable GitHub Pages in repository settings
  • Wait for your site to be published (may take a few minutes)

Step 3: Send Registration Email

Copy this email template:

To: subdomain@hujowe-serwery.online

Subject: Subdomain Registration Request

Hello,

I would like to register the subdomain:
My GitHub repository URL is: [Please insert your GitHub Pages URL here]
(Format: https://yourusername.github.io)

Thank you!

Note: After sending the email, please allow 1-2 days for your subdomain to be registered and activated.
